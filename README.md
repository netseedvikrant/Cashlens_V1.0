# 💎 Cashlens: AI-Powered Personal Expense Tracker

**Cashlens** is a modern, high-performance financial tracking application designed for the mobile-first era. Built with **React**, **Tailwind CSS**, and **Groq AI**, it provides users with deep insights into their spending habits through a beautifully crafted glassmorphism interface.

## ✨ Key Features

### 🧠 **AI Spending Coach**
Receive professional financial advice generated by **Llama-3.3-70B** via the **Groq API**. The AI analyzes your categories, recognizes essential versus luxury spending (like Health vs. Entertainment), and provides actionable one-liners to help you optimize your budget.

### 📊 **Interactive Analytics Dashboard**
- **Dynamic Charting**: Real-time spending breakdown using **Recharts**.
- **Date Range Filters**: Filter your history between specific dates to see historical trends.
- **Smart Totals**: Automatic recalculation of "Total Spent" based on active filters.

### 📱 **Mobile-First Modular UI**
- **375px Audit**: Fully responsive design with zero horizontal scrolling on phone screens.
- **Inline Editing**: Modify expense amounts, categories, or notes directly within the list or mobile cards.
- **Modern Notifications**: Toast-based feedback system for a non-intrusive user experience.

### 📂 **Data Management**
- **Persistence**: 100% Offline-first using `localStorage`—no backend login required.
- **CSV Export**: Download your currently filtered records as a `.csv` file for professional accounting or spreadsheet work.

## 🚀 Getting Started

### **Prerequisites**
- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- A **Groq API Key** (Get one for free at [Groq Console](https://console.groq.com/))

### **Installation**
1. **Clone the repository**:
   ```bash
   git clone https://github.com/netseedvikrant/Cashlens_V1.0.git
   cd Cashlens_V1.0
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your API key:
   ```env
   VITE_GROQ_API_KEY=your_groq_api_key_here
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```

## 📖 How to Use
1. **Log an Expense**: Enter the amount, select a category, and add an optional note.
2. **Analyze**: Use the **Pie Chart** to see which categories are dominating your budget.
3. **Filter**: Use the **Date Range** or **Category** filters to isolate specific spending trends.
4. **Get Insights**: Click **"Ask AI for Insights"** to receive professional-grade financial coaching.
5. **Export**: Click **"Export CSV"** to download your data for external use.
